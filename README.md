# 꾸준히 성장하는 개발자 소성민입니다.

실제 서비스 운영 경험을 바탕으로 사용자와 서비스에 깊이 공감하며 개발하는 백엔드 개발자입니다.
Java를 기반으로 개발을 시작했으며, JavaScript와 TypeScript를 활용한 실서비스 개발 및 배포 경험을 쌓았습니다.

현재는 **Go와 Kotlin을 중심으로 실서비스를 개발·운영**하며 백엔드 역량을 확장하고 있습니다.
또한 AI 기술과 최신 개발 도구를 적극적으로 탐구하며, 개발 과정에서 **AI를 활용한 구현, 자동화, 생산성 향상**을 실무에 적용하고 있습니다.
